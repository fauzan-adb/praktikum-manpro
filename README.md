## Webhook Github?
Webhook merupakan Callback Function yang akan berjalan setelah mendapatkan triger tertentu oleh Action atau Event. Di Github sendiri fungsi webhook adalah untuk mengirim request menggunakan URL yang sudah di tentukan di dalam konfigurasi dari Webhook itu sendiri.
